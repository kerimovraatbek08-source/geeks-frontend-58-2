Что такое Frontend?
Frontend (фронтенд) — это внешняя, клиентская часть веб-сайтов или приложений, с которой взаимодействует пользователь. Она включает визуальный дизайн, верстку, анимацию и элементы управления (кнопки, формы), создаваемые с помощью HTML, CSS и JavaScript. Основная цель — сделать интерфейс удобным, адаптивным и быстрым.
Что делает HTML?
HTML — это язык разметки, который создает структурный каркас веб-страниц, определяя расположение текста, изображений, ссылок, видео и других элементов. Он говорит браузеру, как именно отображать контент, преобразуя набор тегов в структурированный документ, понятный пользователю.
Что такое CSS?
CSS — это язык для описания внешнего вида веб-страниц (их оформления, дизайна), работающий совместно с HTML, который отвечает за структуру контента; CSS определяет цвета, шрифты, отступы, расположение элементов и адаптивность, делая сайт красивым и удобным. Если HTML — это скелет сайта, то CSS — это его одежда и макияж.
Что такое js?
js это основной язык программирования для создания интерактивных веб-страниц, который добавляет динамические функции, такие как анимации, валидация форм и обновление контента, а также используется для разработки серверной части (Node.js) и мобильных приложений
Что делает React?
React — это JavaScript-библиотека от Meta, предназначенная для быстрого и эффективного создания интерактивных пользовательских интерфейсов
